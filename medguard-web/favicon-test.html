Current Favicon

If you can see the MedGuard SA shield logo above, the favicon is working correctly!

If the favicon is still showing the Vue logo:

  1. Hard refresh the page: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  2. Clear browser cache for this site
  3. Check browser tab - the favicon should show the MedGuard shield
  4. Try incognito/private mode to bypass cache
  5. Wait a few minutes - some browsers cache favicons aggressively

Favicon Files

  • SVG Favicon: /favicon.svg - Vector format, scales perfectly
  • PNG Favicons: /favicon-32x32.png and /favicon-16x16.png - Raster formats for older browsers
  • Apple Touch Icon: /apple-touch-icon.png - For iOS devices
  • Web Manifest: /site.webmanifest - PWA configuration
